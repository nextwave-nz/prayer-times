Manual Mode Workflow (Recommended)

Every Month — 5 Minutes

1
Download the WMA monthly PDF from the masjid.
2
In Extract Data: select mosque, year, month. Upload the PDF.
3
Click Prepare Extraction. Copy the prompt shown.
4
Open Claude.ai, start a new chat, attach the PDF, paste the prompt, send.
5
Copy the JSON response from Claude.ai and paste it back into the admin.
6
Click Parse JSON → review table → Generate Code → Copy Code.
7
In index.html on GitHub, replace the old month array with the new one. Update resolveIqamas([...]) to include the new variable. Push — done! 🎉
⚡
Auto Mode (if CORS allows)

Auto Mode Setup

1
Select Auto Mode in the mode selector and enter your API key from console.anthropic.com.
2
If it shows "Failed to fetch" — your browser is blocking the API call due to CORS. Switch to Manual Mode instead.
3
Auto mode may work if you open admin.html from a local computer rather than from GitHub Pages.
