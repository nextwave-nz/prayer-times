# Adhan Audio Files
   
   This folder contains the Adhan MP3 files used by the prayer times app.
   
   - fajr.mp3 — Fajr Adhan (includes "as-salatu khayrun min an-nawm")
   - standard.mp3 — Adhan for Dhuhr, Asr, Maghrib, Isha
   
   Source: Internet Archive — Public Domain Mark 1.0
   https://archive.org/details/adhan.recordings.from.doha.qatar
